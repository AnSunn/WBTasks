Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
test() вернет 2, потому что в этой функции используется именнованная переменная x.
В данном случае отложенная функция имеет доступ к именованному возвращаемому значению x и может изменять его после того,
как функция c завершит выполнение.
то есть в defer попадет значение 1 и увеличится на единицу, в результате будет 2

anothertest() вернет 1. 
В приведенном примере выражение x вычисляется на момент вызова defer, но сам вызов анонимной функции откладывается до завершения функции anothertest().
Таким образом, значение x на момент вызова defer равно 0, увеличиваем на 1 и возвращаем это значение.

NB: отложенные функции выполняются в обратном порядке после завершения функции.

```
